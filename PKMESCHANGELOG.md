# Readme

This is the main branch of the reposatory for the PKMES Project. Based of the PokeEmerald-Expansion (currently v1.12.1) with my own modifactions and other feature branches.

# Features

- Reusable TM's
- All Pokemon from Gen 1 to 9 (SV Current)
- All Moves, Abilities, Items etc.. Gen 1 to 9 (SV Current)
- Terastal, Megas, Z-moves, Dynamax

- Expanded Move Tutor (can teach Egg move, Tutor moves and Level-up moves)
- Expanded Move Pools (All mons can learn any move they have been able to in all gens (Some have been changed for ballance e.g. Fissure, No Guard Machamp))
- Expanded Game Corner (more games and mechanics to use (Certain elements have been updated))

- Game Config Options (Dificulty, No Encounters, No Healing etc...)

- Respawnable Hidden Items
- Respawnable Item Balls (Most)

- New UI's
    - Summery Screen
